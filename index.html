Access Your Data Key

Grab the "key" to connect Claude to your database:

  1. Go to your Vercel Dashboard.
  2. Select your new project.
  3. In the top menu, click Storage > Neon.
  4. Copy the Postgres Connection String.

Connect Claude

Copy this block and paste it into Claude (Settings > MCP > Edit Config):

"postgresql": { "command": "npx", "args": [ "-y", "@modelcontextprotocol/server-postgres", "PASTE_URL_HERE?sslmode=require" ] }
