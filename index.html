Retrieve Data Key

Your store is live! Now grab the unique key to connect Claude AI to your database.

  1. Go to Vercel Dashboard > Select Project.
  2. Go to Storage > Neon.
  3. Copy the Postgres Connection String.

Connect to Claude

Open Claude Desktop (Settings > MCP > Edit Config) and paste this block:

"postgresql": { "command": "npx", "args": ["-y", "@modelcontextprotocol/server-postgres", "PASTE_URL_HERE?sslmode=require"] }
