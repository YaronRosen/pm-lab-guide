Access Your Data Key

Once deployment is finished, we need the "key" to connect the AI to your database:

  1. Go to your Vercel Dashboard.
  2. Click on your new project.
  3. In the top menu, click the Storage tab.
  4. Select Neon and copy the Postgres Connection String.

Connect to Claude AI

Copy this configuration block and paste it into your Claude Settings (Settings > MCP > Edit Config):

"postgresql": { "command": "npx", "args": [ "-y", "@modelcontextprotocol/server-postgres", "PASTE_URL_HERE?sslmode=require" ] }

Note: Replace 'PASTE_URL_HERE' with your key and save the file.

Final Lab Test

Open Claude and start a new chat. Run this command to verify the connection:

"Connect to my database and list the names of the top 5 products in my store."
