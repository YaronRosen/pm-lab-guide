Step 2: Get Your Data Key

The store is live! Now, let's grab the connection key so the AI can analyze your data:

  1. Go back to the Vercel tab and click Storage.
  2. Select your Neon database icon.
  3. Copy the Postgres Connection String (the long link starting with postgres://).

Step 3: Connect the Analyst

Open the Claude Desktop app. Go to Settings > MCP > Edit Config. Paste this code block:

"postgresql": { "command": "npx", "args": [ "-y", "@modelcontextprotocol/server-postgres", "PASTE_URL_HERE?sslmode=require" ] }

Remember: Replace 'PASTE_URL_HERE' with your unique key.
