2 Get the Connection URL

After deployment is finished:

  1. Go to the Storage tab in Vercel.
  2. Click on your Neon database.
  3. Copy the "Postgres Connection String" (it starts with postgres://).

3 Setup Claude Analyst

Open Claude Desktop > Settings > MCP > Edit Config. Paste this code inside "mcpServers":

"postgresql": {
  "command": "npx",
  "args": [
    "-y",
    "@modelcontextprotocol/server-postgres",
    "YOUR_POSTGRES_URL_HERE?sslmode=require"
  ]
}

Final Test: Open a new chat in Claude and ask:
"List all product categories in my database and count how many products are in each."

Having trouble?

  • Claude won't connect? Make sure you restarted the Claude app completely after saving the config file.
  • Database Error? Double-check that you added ?sslmode=require at the very end of your connection string.
  • Vercel stuck? Ensure you have a Neon account linked; sometimes a quick refresh of the Vercel page helps.
