Start Your AI Lab

This setup will create a live e-commerce store and link it to Claude AI for advanced product analysis.

Vercel Login

Vercel is the cloud platform that hosts your store. Log in now to ensure a smooth deployment.

Login to Vercel

GitHub Storage

GitHub stores the code "blueprints" for your lab. Ensure you are signed in.

Login to GitHub

Database Infrastructure

We will now create the Postgres Database that will hold all your products, customers, and orders.

Action Required:
1. Go to the Vercel Storage Dashboard.
2. Click Create Database and select Neon (Postgres).
3. Once created, click Connect and copy the Postgres Connection String (keep it for Step 5).

Deploy Lab Environment

With the infrastructure ready, we will now launch the store. Click the button below to trigger the deployment.

Vercel Prompts:
1. Look for the variable MEDUSA_ADMIN_ONBOARDING_TYPE.
2. Enter the value: nextjs.
3. Click the Deploy button.

Claude AI Connection

To analyze data, Claude needs to speak to your database. Paste this config into Claude Desktop (Settings > MCP > Edit Config).

"postgresql": { "command": "npx", "args": ["-y", "@modelcontextprotocol/server-postgres", "PASTE_URL_HERE?sslmode=require"] }

Replace PASTE_URL_HERE with the string you copied in Step 3.

System Test

Verify your lab is operational. Open a new chat in Claude and ask:

"Connect to my database and tell me the names of the top 3 products in stock."

Setup Successful

Your AI Lab is live! Your instructor environment is ready for class.
