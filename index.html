Welcome! Let's build your lab.

Before we start, let's personalize your setup.

Step 1: Vercel Login

Vercel will host your website and database. Please make sure you are logged in.

1. Click the button below to open Vercel.
2. Sign in with Google (recommended).
3. Once logged in, come back here.
Login to Vercel

Step 2: GitHub Login

GitHub will store your lab's code privately. Please ensure you are connected.

1. Click the button below to open GitHub.
2. Sign in or create an account (using Google/Email).
3. Once logged in, come back here.
Login to GitHub

Step 3: Claude AI Ready

We will use Claude as your Product Analyst. Make sure you have the tools ready:

  • Ensure Claude Desktop is installed on your computer.
  • Make sure you have an active Claude account.
Download Claude Desktop

Step 4: Launch Your Store

Now that everything is ready, let's deploy your store! This takes about 3 minutes.

Step 5: Get Your Data Key

Store is live! Now, grab the key for the AI:

  1. Go to your Vercel Dashboard.
  2. Click on your new project.
  3. Click the Storage tab in the top menu.
  4. Select Neon and copy the Postgres Connection String.

Step 6: Connect Claude

Paste this code into your Claude Desktop config (Settings > MCP > Edit Config):

"postgresql": { "command": "npx", "args": [ "-y", "@modelcontextprotocol/server-postgres", "PASTE_URL_HERE?sslmode=require" ] }

Replace 'PASTE_URL_HERE' with your key and Save.
